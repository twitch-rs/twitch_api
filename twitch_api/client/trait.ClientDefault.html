fn default_client_with_name( product: Option<HeaderValue>, ) -> Result<Self, Self::Error>

Constructs Self with sane defaults for API calls and oauth2 and setting user-agent to include another product

Specifically, one should

  • Set User-Agent to {product} twitch_api/{version_of_twitch_api} (+https://github.com/twitch-rs) (According to RFC7231) See TWITCH_API_USER_AGENT for the product of this crate
  • Disallow redirects
§Notes

When the product name is none, this function should never fail. This should be ensured with tests.
