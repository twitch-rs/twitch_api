§prefix: String

The name portion of the Cheermote string that you use in chat to cheer Bits.

The full Cheermote string is the concatenation of {prefix} + {number of Bits}. For example, if the prefix is “Cheer” and you want to cheer 100 Bits, the full Cheermote string is Cheer100. When the Cheermote string is entered in chat, Twitch converts it to the image associated with the Bits tier that was cheered.

§bits: i32

The amount of bits cheered.

§tier: i32

The tier level of the cheermote.
